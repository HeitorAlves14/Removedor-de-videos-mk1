# Removedor-de-videos-mk1
Um script simples que remove vídeos do youtube.

Como usar:
1- Abra o youtube, va em playlists e assistir mais tarde.
2- Precione F12 e cole o script no console.
Pronto, agora é só esperar que ele remove todos os vídeos. Se quer que ele faça a ação mais rápido, apenes altere os dos valores finais, quanto menor for o valor mais rápido.
Eu não testei um limite de quanto rapido ele pode ficar, mas caso dé qualquer erro volte aos valores 500 e 1000.
